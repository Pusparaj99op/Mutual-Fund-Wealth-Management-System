──────────────────────────────────────────
  1. Navigate to folder:
     cd palloti

  2. Run auto-setup:

     ./quickstart.sh

  3. When backend starts, open new terminal:
     source venv/bin/activate
     streamlit run app/dashboard.py

  4. Open browser:
     http://localhost:8501
